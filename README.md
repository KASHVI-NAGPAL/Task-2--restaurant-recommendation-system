Restaurant Recommendation System

A Machine Learning-based Restaurant Recommendation System developed using Python and Content-Based Filtering techniques. This project recommends restaurants to users based on their preferences such as cuisine type and price range.

Project Objective

The main objective of this project is to build a recommendation system that suggests restaurants similar to user preferences using restaurant features and cosine similarity.

Features
Data preprocessing and cleaning
Handling missing values
Encoding categorical variables
Feature engineering
Content-based filtering
Cosine similarity recommendations
Personalized restaurant suggestions
Technologies Used
Python
Pandas
NumPy
Scikit-learn
Machine Learning Concepts Used
Content-Based Filtering
Feature Extraction
Count Vectorization
Cosine Similarity
Workflow
Load and preprocess the dataset
Handle missing values
Encode categorical features
Create combined feature columns
Convert text data into vectors
Calculate cosine similarity
Recommend restaurants based on user preferences
Sample Input
Cuisine Preference: North Indian
Price Range: Medium
Sample Output

Recommended restaurants with similar cuisine and pricing preferences.
